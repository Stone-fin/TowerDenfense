# 塔防模拟

## 实现功能
一个塔防游戏，可以手动游玩，也可以提供接口用来训练和演示AI训练。

## 游戏规则
游戏地图为n*n的方形网格，游戏开始，玩家决策在地图中选择３＊３的网格放置中央能量塔。中央塔可以产生基础能量功率输出和基础物资生产，后续建造相应的建筑，生产类、功能类、战斗类。不同的建筑的占用面积不同，功能不同，消耗能量速度不同，建造所需物资不同。

游戏开始后，怪物巢穴会随机生成一个在玩家可操作地图边界外，不定时生产怪物，怪物会攻击玩家的建筑及中央塔随着时间推移生产的怪物会越来越强，因此最终玩家肯定会失败。

在注定失败的情况下，玩家的目标就是尽可能得存活更长时间。
